Goal: sell a fixed-scope CRM Import Rescue project starting at `$199`, with a `$49` sample audit.

Use this list to find partners and buyers who already feel CRM import pain. Do not bulk spam. Send short, specific notes and only follow up once.

## First Message

```text
Quick question: do you have a HubSpot, Salesforce, or Airtable import coming up with messy contacts?

I built a small CRM Import Rescue tool that finds duplicate people/companies, bad emails, inconsistent phone numbers, unmapped fields, and missing required columns before upload.

Fixed-scope cleanup starts at $199, with a $49 sample audit. If you send a sample export, I can tell you what will break before you upload it.
```

## Follow-Up

```text
One useful way to test this: send me a 20-row anonymized sample. I will return a short import blocker report so you can decide if the full cleanup is worth doing.
```

## Notes

- Never ask for credentials.
- Do not accept lists that the prospect does not have permission to use.
- For sensitive or regulated data, require a proper agreement before handling it.
- For agencies, position this as a subcontracted cleanup sprint, not a competing CRM implementation service.
